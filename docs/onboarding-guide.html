✅ Setup Checklist

1

Select Categories

Go to Categories page and select all service categories your company offers (e.g., Automobile Washing, Electrician).

2

Add Services

Go to Services page and add all services with names, prices, and descriptions. These appear to users when booking.

3

Add Workers

Go to Technicians page and add all your workers with names, phone numbers, and assigned services.

4

Set Availability

Go to Calendar / Slots page and set your working hours. Block dates when you're not available.

5

Start Accepting Bookings

Once setup is complete, users can find your company on the app and start booking services!

📊

Real-time Dashboard

OTP Verification

Secure 6-digit OTP system ensures worker arrival verification before work begins.

Worker Management

Smart Scheduling

Revenue Tracking

Feedback System
